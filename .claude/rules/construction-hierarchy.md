# 工種階層（工事写真分類体系）

## 階層構造

```
写真区分
└── 工種
    └── 種別
        └── 細別
            └── 作業内容
```

## フィールド対応

| フィールド | 日本語 | Level | 例 |
|-----------|--------|-------|-----|
| photoCategory | 写真区分 | 1 | 施工状況写真 |
| workType | 工種 | 2 | 舗装工 |
| variety | 種別 | 3 | 舗装打換え工 |
| detail | 細別 | 4 | 表層工 |
| - | 作業内容 | 5 | 舗設状況 |

## 具体例：舗装工

```
施工状況写真
└── 舗装工
    ├── 舗装打換え工      ← 種別
    │   ├── 表層工        ← 細別
    │   │   ├── 舗設状況
    │   │   ├── 初期転圧状況
    │   │   └── 施工完了
    │   └── 上層路盤工    ← 細別
    │       ├── 不陸整正状況
    │       └── 転圧状況
    └── 未舗装部舗装工    ← 種別
        └── ...
```

## よくある間違い

❌ **絶対にやってはいけない**:
```
種別: 表層工      ← 間違い！表層工は細別
細別: 舗設        ← 間違い！舗設は作業内容
```

✅ **正しい**:
```
種別: 舗装打換え工
細別: 表層工
```
