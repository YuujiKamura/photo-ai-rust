# Task: 測点候補リスト機能

## Description

測点入力時に候補リストを表示し、オートコンプリートで入力を効率化する。候補はマスタファイルまたは既存入力から取得。

## Acceptance Criteria

- [ ] 入力済み測点から候補リストを自動生成
- [ ] マスタファイルから測点候補を読み込み（オプション）
- [ ] Tab補完またはインクリメンタル検索で候補選択
- [ ] 候補がない場合は自由入力を許可

## Technical Details

```rust
use dialoguer::FuzzySelect;

let stations: Vec<&str> = collect_existing_stations(&results);

let selection = FuzzySelect::new()
    .with_prompt("測点を選択（または入力）")
    .items(&stations)
    .interact_opt()?;
```
